
# 💼 Job Application Tracker

A full-stack Job Application Tracker to help users manage, monitor, and organize job applications efficiently.

---

## 🌐 Tech Stack

- **Frontend**: Angular 17
- **Backend**: Node.js, Express
- **Database**: MySQL
- **Authentication**: JWT
- **Styling**: CSS (Dark Mode support)

---

## ✨ Features

- 🔐 Signup/Login authentication
- ➕ Add job applications
- ✏️ Edit application status
- 🗑️ Delete application
- 📊 Summary stats (total applied, interviews, offers)
- 🌓 Dark mode UI
- 🚫 Protected routes

---

## ⚙️ Project Setup

### ✅ Backend Setup


- cd backend
- npm install
- node index.js

### ✅ Frontend Setup


- cd frontend
- npm install
- npm start

### ✅ DB Setup


- run query
